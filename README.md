# VideoRadar - 유튜브 영상 리서치 레이더

YouTube Data API v3를 활용하여 키워드별 영상을 분석하고, 최적의 영상을 수집할 수 있는 대시보드입니다.

## 🚀 주요 기능
- **실시간 영상 분석**: 조회수, 구독자 대비 성과도, 기여도 자동 계산
- **스마트 필터링**: 쇼츠 제외, 조회수 범위, 등급별 필터링 제공
- **검색 히스토리**: 과거 검색 기록 관리 및 캐시를 통한 빠른 데이터 조회
- **모바일 최적화**: 어떤 기기에서도 편리한 반응형 UI

- **Backend**: Node.js, Express

- **API**: YouTube Data API v3
